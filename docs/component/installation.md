# 安装

## 使用包管理器

我们建议您使用包管理器（如 NPM、Yarn 或 pnpm）安装 Element Plus，然后您就可以使用打包工具，例如 Vite 或 webpack。
::: code-group
```js{1}  [npm]
npm install ns-node -D
```
```js  [yarn]
yarn add ns-node -D
```
```js  [pnpm]
pnpm install ns-node -D
```
:::
